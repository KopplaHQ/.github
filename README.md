# .github

Organization-level GitHub configuration for [KopplaHQ](https://github.com/KopplaHQ).

The `profile/README.md` is the public organization profile displayed at [github.com/KopplaHQ](https://github.com/KopplaHQ). It is automatically updated daily with the latest blog posts from [kopplamarketing.com](https://kopplamarketing.com/blog/).
